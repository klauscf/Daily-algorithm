/*******************************************
*	快速排序时间复杂度问题 
*	时间复杂度：最差的时间复杂度O(N^2)，
*				平均复杂度为O(NlogN) 
*	原理：快排每次交换为跳跃式，总体交换次数减少，
*		基于一种二分的思想 	 
********************************************/ 

它的基本思想是：通过一趟排序将要排序的数据分割成独立的两部分，
              其中一部分的所有数据都比另外一部分的所有数据都要小，
              然后再按此方法对这两部分数据分别进行快速排序，
              整个排序过程可以递归进行，以此达到整个数据变成有序序列。
